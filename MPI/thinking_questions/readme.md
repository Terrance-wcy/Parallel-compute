直接运行run.sh脚本文件可依次编译并执行下述程序
1.矩阵乘积

代码：juzhen.c 

生成文件：juzhen

运行方式：mpirun -np 4 juzhen 

运行效果：随机生成两个4*4矩阵计算两个矩阵乘积

2.任意阶矩阵乘积

代码：juzhen2.c

生成文件：juzhen2

运行方式：juzhen2

运行效果：首先输入第一个矩阵行数和列数，接着输入第二个矩阵行数和列数，接着输入第一个矩阵每个位置的元素，接着输入第二个矩阵每个位置元素，计算乘积，此程序也可计算矩阵向量相乘（行数设为1即可）,在脚本中使用的是2*1的元素为2矩阵和1*3的元素为3矩阵，相乘得到2*3的元素为6的矩阵，如需修改，可以直接修改run.sh文件。

3.雅各比迭代

代码：jacobi.c

生成文件：jacobi

运行方式：mpirun -np 4 juzhen 

运行效果：创建8*8矩阵，矩阵第一行和第一列设为初始值5，开始迭代并输出结果
